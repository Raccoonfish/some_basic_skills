<h2 align="center">postgres.sh</h2>
Этот скрипт устанавливает и настраивает PostgreSQL, создаёт пользователя john со всеми правами на пространство public, чтобы он мог добавлять таблички.  
Также редактируются конфигурационные файлы pg_hba.conf (замена проверки локального пользователя на проверку пароля) и postgresql.conf (возможность подключаться к БД из локальной сети).
